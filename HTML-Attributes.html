    Syntax of an HTML Attribute

    All attributes 2 parts Se Mil kar Bane Hote Hai |

    1. Attribute Name.
    2. Attribute value.
    1. Attribute Name : Name एक Property है , जिसे आप Set करना चाहते है | For Example:- Image को Extra Configure करने के लिए src Attribute का इस्तेमाल किया जाता है | तो यहाँ पर Image Tag के Attribute का Name src है | जिसका इस्तेमाल हम Image को Extra Configure करने के लिए करेंगे |
    2. Attribute Value :- जो आप Property के द्वारा उस Element मै Set करना चाहते है | Value को हमेशा quotation(” “) के अंदर लिखा जाता Hai | उप्पर दिए गए Image और Example मै देख सकते है | Attribute की Value URL है, जहा पर आप Image का URL set करने पर Web Page मै Image Show होने लगती है |

    For Example:-

     <html>
  <head>
   <title>HTML ATTRIBUTE IN HINDI</title>
  </head>
  <body>
   <img src="https://NdrItTechProgramming.com" alt="Image Error">
  </body>
 </html>

    अब तक तो थोड़ा बहुत आपको कुछ समँझ जरूर आया होगा की HTML attribute किया होता है ? और इसका इस्तेमाल हम HTML Elements को Additional Ingormation Provide करने के लिए कैसे कर सकते है |

    HTML Attribute List in Hindi

    अब हम HTML Attribute को One By One देखते है जो ज्यादातर इस्तेमाल होते है |

    1.href Attribute

    Html Link को से <a> define किया जाता है । <a> tag मैं link को specified करने के लिए href attribute का इस्तेमाल होता है | 

    	<a href="https://masterprograming.com/">This Is Link Tag</a>

    2. src Attribute

    HTML मैं Image Add करने के लिए हम <img> Tag का इस्तेमाल किया जाता है | src img tag का attribute है | Image का source address set करने के लिए src attribute का इस्तेमाल किया जाता है | 

    	<img src="https://NdrItTechProgramming.com" alt="Image" width="500" height="600">

    3. Style

    Style Attribute का इस्तेमाल किसी Tag पर Inline CSS Rule Apply करने के लिए किया जाता है | 

    	<p style="color:red">This Is Paragraph Tag</p>

    4. Width and height Attribute

    HTML मैं Image का आकर यानि Size Set करने के लिए Size attribute का इस्तेमाल होता है | size attribute की help से image की height or width set की जाती है | 

    <img src="https://NdrItTechProgramming.com" alt="Image" width="500" height="600">

    5. ALT attribute

    ALT Attribute का इस्तेमाल एक Image के बारे मैं Alternative text लिखने के लिए किया जाता है | यह Text Picture के Show नहीं होने पर दिखाई देता है| 

    	<img src="https://NdrItTechProgramming.com" alt="Image" >

    6. title Attribute

    Title Attribute किसी element के बारे मैं extra information provide करता है । जब mouse उस element के उपर लिया जाता है, जिसमे title attribute इस्तेमाल हुआ है, तो set की गई Imformation tool-tip text के रूप मै show हो जाती है | 

       <p title="I'm a tooltip">
	This Is Paragraph Tag
   </p>

    7. lang Attribute

    इस Attribute का इस्तेमाल Content की Language को Define करने के लिए किया जाता है. इस attribute को हम HTML Element पर Apply करते है | 

    <! DOCTYPE html>
<html lang="hi-ind">
  <head>
    <title>HTML ATTRIBUTE IN HINDI</title>
  </head>
</html>

    8. Class

    Class Attribute भी ID की तरत ही होता है . इसका इस्तेमाल एक Element की Class Define करने के लिए किया जाता है |
