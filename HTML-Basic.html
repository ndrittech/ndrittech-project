    HTML Basic In Hindi

    HTML Basic मैं HTML Code Start करने से पहले HTMLमैं Start <html>Or ends </html>. इसके साथ करना है |

    HTML मैं Comments लिखने के लिए आप <!-- Comments -->. ये इस्तेमाल करना होगा | 

    	<html>
		<!-- Comments -->
	</html>

    Basic In Hindi Documents मैं जितना भी Visible यानि दिखाई देने बाला Part होता है वो between <body> and </body>. इसके अंदर आता है | 

    	<html>
		<body>
			This Is Body Tag
		</body>
	</html>

    Html Basic Tags in hindi

    HTML Heading Tags :- HTML मैं Heading 6 Tags Type की होती है | इन्हे हम h1 , h2 , h3 , h4 ,h5 , h6. से Define करते है | आप इनके निचे दिए Program Execute करके Differnce देख सकते है | 

    	<html>
		<body>
			<h1>This Is Heading Tag</h1>
			<h2>This Is Heading Tag</h2>
			<h3>This Is Heading Tag</h3>
			<h4>This Is Heading Tag</h4>
			<h5>This Is Heading Tag</h5>
			<h6>This Is Heading Tag</h6>
		</body>
	</html>

    HTML Paragraphs Tag :- HTML मैं Paragraph tag इसलिए इस्तेमाल होता है ताकि हम Paragraphs लिखे सकते | इन्हे हम <p>. से Define करते है | आप इसका Example निचे दिए Program को Execute करके देख सकते है | 

    	<html>
		<body>
			<p> This Is Paragraph Tag </p>
		</body>
	</html>

    HTML Link tag :- HTML मैं Link tag इसलिए इस्तेमाल होता है ताकि हम किसी link को link कर सके | इसे हम <a>.से Define करते है | आप इसका Example निचे दिए Program को Execute करके देख सकते है | 

     <html>
	<body>
	   <a href="https://masterprograming.com/">This Is Body Tag</a>
  	</body>
 </html>

    आप ऊपर Program को देख सकते है हमने उसमे href attribute इस्तेमाल किया है इसके बारे मैं हम आगे और Study करेंगे | वैसे href attribute Additional Information Provide करता है |

    Html Basic Tags in hindi

    HTML Buttons Tags:- HTML मैं Button tag इसलिए इस्तेमाल होता है ताकि हम Button बना सके | इसे हम <button>tag . से Define करते है | आप इसका Example निचे दिए Program को Execute करके देख सकते है | 

      <html>
	<body>
	  <button>This Is Body Tag</button>
	</body>
  </html>

    HTML Images Tags :-HTML मैं Image tag इसलिए इस्तेमाल होता है ताकि हम Images को Add कर सके | इसे हम <img>tag. से Define करते है | आप इसका Example निचे दिए Program को Execute करके देख सकते है | 

    	<html>
	  <body>
	    <img src="images/img.jpg" alt="Danish" width="250" height="150">
	  </body>
	</html>

    Source File को Load करने के लिए ( src ) का इस्तेमाल होता है , alternative text (alt) के text लिखने के लिए इस्तेमाल होता है और अगर image किसी वजह से browser मई show नहीं होता है तो alt text मई जो लिखा होता है वही show होता है | , Image को Size देने के लिए width, and height का इस्तेमाल होता है |

    HTML Line Break Tags :- HTML मैं Line Break Tag इसलिए इस्तेमाल होता है ताकि एक लाइन के बाद Break लगा सके | इसे हम <br> tag . से Define करते है | आप इसका Example निचे दिए Program को Execute करके देख सकते है | 

      <html>
	<body>
	  <p> This Is Paragraph <br> Line Break Tag </p>
	</body>
  </html>

    HTML List Tags :- HTML मैं List Tag इसलिए इस्तेमाल होता है जब हम List के Order Data को Show करते है | इसे हम <ul> (unordered/bullet list) और <ol> (ordered/numbered list) tag And इन्हे हम <li> tags के अंदर लिखते है | से Define करते है | आप इसका Example निचे दिए Program को Execute करके देख सकते है | 

    <html>
  <body>
	<ul>
	  <li>Download</li>
	  <li>Now MasterPrograming</li>
	  <li>Apps</li>
	</ul>
  
	<ol>
	  <li>Download</li>
	  <li>Now MasterPrograming</li>
	  <li>Apps</li>
	</ol>
 </body>
</html>
