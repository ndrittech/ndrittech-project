    इस IDE को आप यहां Click करके Sublime Text Editor Download कर सकते है | इसे Install करने के निचे steps दिए गए Steps Follow करे |

    1. Download —–> Sublime Text Editor

    2. HTML Sublime Text को आपको Download करने के बाद आप उस File के ऊपर Right Click करके Run as administrator पर Click करना होगा जैसे की आपको ऊपर दिए गए Image को दिखाया गया है |

    3. जब आप Run as Administrator पर click करने के आपको उपर दिए image में केवल आपको NEXT press करना है | अगर आप कोई और drive में इस Sublime Text Software को install करना चाहते है तो आप Drive को Change कर सकते है |

    4. यहां पर भी केवल आपको NEXT press करना है |
