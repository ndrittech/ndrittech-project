    HTML Elements In Hindi:- HTML Element or HTML Tag मैं क्या Difference है – Different between HTML Tag and HTML Element? HTML Element के कितने Types होते है – Types of HTML Elements? |

    HTML Element Kiya Hai ? – What is an HTML Elements?

    HTML Element In Hindi , HTML Tags और Content का Combination होता है , यानि के Opening HTML Tag ( <opening> ) और Closing HTML Tag ( </closing> ) तक सब कुछ एक HTML Element ही होता है. एक Start HTML Tag, End HTML Tag और इनके बीच मैं जो Content का Group होता है उसे ही हम HTML Element कहते है.
    HTML Element In Hindi को Understand करने के लिए हमने कुछ Example तैयार किये है | जिसे Read करके आप HTML Element को अच्छी तरह से समझ पाएंगे

    HTML Elements In Hindi

    HTML Element Mainly 2 Types के tag होते है |

    • Container tag
    • Empty tag & Non Container

    1. Container tag

    Start tag Element content End tag
    <h1> My First Heading </h1>
    <h2> This Is Heading 2 </h2>
    <p> My first paragraph. </p> 
    	<html>
		<body>
			<p>My first paragraph.</p>
		</body>
	</html>

    HTML Elements In Hindi

    ऊपर दिए गए Example मैं 3 HTML Elements है. उन्हें हम One By One HTML Element को Understand करते है.

    1. HTML Element का HTML Document का Root Element होता है. इस Element से Browser को पता चलता है की यह एक HTML Document है. Opening <html> Tag, और HTML Elements (Body, Heading or Paragraph), और Closing </html> Tag है |

    2. Body Element मैं HTML Document का Visible Part लिखा जाता है. Opening <body> Tag, और Other HTML Elements (Heading Or Paragraph), और Closing </body> Tag से एक Body Element का निर्माण (Construction) होता है |

    3. Paragraph Element का उपयोग HTML Document मैं Paragraph लिखने के लिए किया जाता है. Opening <p> Tag, Paragraph Text, or Closing </p> Tag से एक Paragraph Element का निर्माण (Construction) होता है |

    ये closing tag, Opening tag जैसे ही होते है. लेकिन Closing Tag के साथ (/) का इस्तेमाल किया जाता है इसलिए इन्हे हम Container tag कहते है |

    2. Empty Tag or non container tag

    Empty Tag को अकेले ही इस्तेमाल किया जाता है इसमें हम Closing Tag (</> ) का इस्तेमाल नहीं करते है | For Example :- <hr> tag का इस्तेमाल Horizontal Line Print करने के लिए होता है | <hr> और <br> Tag का कोई भी Closing Tag नहीं होता है |
