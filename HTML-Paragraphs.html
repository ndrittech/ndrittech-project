    HTML Tag Reference

    Ham Is Post Mai Niche Diye Gaye Tag Ke Bare Mai Read Karne Wale Hai |

    Tag Description
    <p> Defines a paragraph
    <br> Inserts a single line break
    <pre> Defines pre-formatted text

    How to Create HTML Paragraphs.

    Paragraph Define Karne Ke Liye Aapko Opening <p> Tag Duvara Define Kiya Jata Hai . Or Closing </p> Tag Duvara Closed Kiya Jata Hai |

    Aap HTML Code Ke Paragraphs Mai Kuch Space Bhi Dete ho to bhi Vah Browser Mai Show Nhi Hoga | Kyoki Browser Vah Space Ignore Kar Deta Hai | Yadi Aap Words Mai Space Ko Change Karna Chahte Hai to Aapko CSS Ko Learn Karna Hoga |

    Syntax :

       <p> This Is Paragraph Tag </p>

    Example:

    <!DOCTYPE html>
<html>
  <head>
   <title>heading attribute example</title>
  </head>
  <body>
   <p> This Is Paragraph Tag </p>
  </body>
 </html>

    Paragraph Alignment

    By Default HTML Paragraph ka Alignment Left Hota hai. Matlab Paragraph Browser Window Mai Left ki taraf se start ho kar right taraf jata hai |

    Agar Aap Paragraph ki Default Alignment ko Align Attribute ke duvara Override Kar Sakte Hai|. HTML Align Attribute Ke Liye Ek Paragraph Ke Liye 4 Alignments Set kiye Ja Sakte Hai.

    • Left Alignment
    • Right Alignment
    • Center Alignment
    • Justify Alignment

    1.Left Alignment

    	<p align="left" This is Left Side Paragraph </p>

    2.Right Alignment

    	<p align="right" This is Right Side Paragraph </p>

    3.Center Alignment

    	<p align="center" This is Center Side Paragraph </p>

    4.Justify Alignment

    	<p align="justify" This is Justify Side Paragraph </p>

    Paragraph Alignment Example

    Hamne Upper Bataye Gaye Sabhi Alignments ko sath likhar ek example teyar kiya hai. Jisme Sabhi Alignments ko alag-alag paragraphs par apply karke dikhaya gaya hai |

    Example:

    <!DOCTYPE html>
<html>
  <head>
   <title>This Is Alignment Example</title>
  </head>
  <body>
	<p> This is Default P </p>
	<p align="left" This is Left Side Paragraph </p>
	<p align="right" This is Right Side Paragraph </p>
	<p align="center" This is Center Side Paragraph </p>
	<p align="justify" This is Justify Side Paragraph </p>
  </body>
 </html>

    HTML Line Breaks In Hindi

    <br> tag in Hindi

    <br> tag ka Istemal Bina Kisi Extra Blank Space liye Paragraph ko Change Karne Ki Facility Provide Karata Hai | Yah Ek Type Ka Empty Tag Hai |

    Line break ek tag Hai । Html Mai Line break tag ko <br> duvara define kiya Jata Hai | Aap Iska Istemal Tab Kar Sakte Hai Jab Aapne New Paragraph Start Nhi Karna Ho Lekin Line Ko Break Karke Shuru se Start Karni Ho |

    Syntax :

       <p> This Is Paragraph <br> this is line break </p>

    Example:

    <!DOCTYPE html>
<html>
  <head>
   <title> Line Break </title>
  </head>
  <body>
   <p> This Is Paragraph <br> this is line break </p>
  </body>
 </html>
